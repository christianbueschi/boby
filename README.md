# Boby

## Organize your browser tabs into collections with drag-and-drop. A clean, fast new tab page for managing your open tabs.

Boby replaces your new tab page with a clean workspace for organizing browser tabs into collections.

How it works:
Your open tabs appear in a sidebar. Drag and drop them into named collections to save them for later. Click any saved tab to reopen it.

Features:

- Drag and drop tabs into collections
- Create, rename, and reorder collections
- Live sync with your open browser tabs
- Edit and rearrange saved tabs across collections
- Export and import your collections as JSON
- Dark theme, clean UI
- All data stored locally — nothing leaves your browser

Built for anyone who works with many tabs and wants a simple way to keep them organized.
